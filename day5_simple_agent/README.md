# Day 5 – Simple AI Agent (REACT vs PREACT Concept)

This project demonstrates the basic idea of an AI Agent that can decide when to use a tool and when to respond directly.

The goal of this exercise is to understand how modern AI systems combine LLMs with external tools to solve tasks.

## Overview

Traditional language models simply generate responses based on prompts. However, AI Agents extend this capability by allowing models to:

- Reason about a problem
- Decide whether a tool is required
- Execute the tool
- Use the result to produce a final answer

This project implements a very simple agent in Python that can:

- Answer normal questions
- Detect mathematical queries
- Use a calculator tool to compute results

## Concepts Covered

- AI Agents
- Tool usage with LLM-style reasoning
- REACT framework (Reason + Act)
- PREACT framework (Plan → Reason → Act)

These concepts are used in modern systems such as:

- LangChain Agents
- AutoGPT
- Tool-enabled LLM assistants

## Project Structure

day5_simple_agent/

### tools.py
